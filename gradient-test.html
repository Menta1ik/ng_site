Профессиональные градиенты для IT/Enterprise продукта

1. Графитовый → Серый
Строгий, универсальный фон для корпоративных и финтех решений. Отлично сочетается с любыми акцентами.
2. Темно-синий → Стальной
Технологичный, современный. Подходит для дашбордов, аналитики, IT-продуктов.
3. Глубокий синий → Индиго
Корпоративный, внушающий доверие. Часто используется в банковских и B2B сервисах.
4. Сапфировый → Синий
Чистый, строгий, отлично подходит для SaaS и enterprise платформ.
5. Графитовый → Синий
Современный, минималистичный. Хорошо смотрится с белыми и голубыми элементами интерфейса.
6. Темно-серый → Стальной
Нейтральный, премиальный. Подходит для серьезных корпоративных решений.
7. Темно-синий → Бирюзовый
Современно, свежо, с акцентом на инновации. Хорошо для IT-стартапов и финтеха.
